# Product Inventory System with Basic Pricing Engine

### Built with:
- Laravel 11
- PHP 8.2^
- Inertia
- Vue 3
- PHP Unit
- Vite
- Tailwind
- A local database
- Valet Plus

### Local Dev
- Clone Repo
- composer install
- npm install
- set up .env
- migrate and seed database
- npm run dev

### Testing
Tests have been set up for the models and routes.

```bash
php artisan:test
```
